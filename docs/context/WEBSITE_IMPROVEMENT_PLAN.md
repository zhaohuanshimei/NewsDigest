# News Digest 网站整改计划

## 📋 总体目标

在保持极简风格的前提下，将 News Digest 提升为一个专业、合规、用户体验优秀的新闻聚合网站。

---

## 🔍 现状分析

### ✅ 现有优势
- 极简清爽的设计风格
- 响应式设计，移动端友好
- RSS 订阅功能
- 归档系统
- 暗黑模式支持
- 快速加载（静态站）
- 来源标注清晰

### ❌ 核心差距

#### 1. 内容展示层面
- [ ] 缺少新闻配图/缩略图
- [ ] 缺少分类/标签系统
- [ ] 缺少热点标识

#### 2. 功能层面
- [ ] **无搜索功能**（高优先级）
- [ ] 无社交分享按钮
- [ ] 无阅读进度/标记功能
- [ ] 无多语言切换

#### 3. 信息透明层面
- [ ] **缺少"关于"页面**（高优先级）
- [ ] 缺少使用说明/帮助
- [ ] 缺少隐私政策
- [ ] 缺少联系方式

#### 4. 技术层面
- [ ] **SEO 元数据不完整**（高优先级）
- [ ] 缺少 sitemap.xml
- [ ] 缺少 robots.txt
- [ ] 无性能监控
- [ ] 无用户分析

#### 5. 用户体验层面
- [ ] 缺少加载状态提示
- [ ] 缺少空状态优化
- [ ] 归档页面交互单薄
- [ ] 缺少快捷键支持

---

## 📐 设计原则（保持极简）

### 核心原则
1. **内容优先**：新闻内容是唯一主角
2. **零干扰**：无广告、无弹窗、无自动播放
3. **性能第一**：保持极速加载
4. **系统字体**：不引入额外字体文件
5. **最小化 JS**：能用 CSS 解决的不用 JS

### 视觉语言
- **色彩**：保持现有的淡雅配色
- **排版**：大量留白，呼吸感
- **动效**：微妙、克制、流畅
- **图标**：使用 SVG，语义化
