# News Digest V2

前后端分离重构版，面向 Web 与后续多客户端扩展。

## 目录

- `apps/web`: 新前端
- `services/api`: 新后端
- `packages/shared-types`: 前后端共享契约
- `packages/ui`: 设计系统与可复用 UI
- `docs/`: 业务背景、设计资料、架构文档、迁移计划
- `legacy-reference/`: 旧项目关键说明的只读参考

## 当前状态

当前仓库已经完成从 V1 到 V2 的主线切换准备：

- V1 已冻结到 `legacy/news-digest-v1`
- 新主线只保留重构所需的文档、设计稿和精选 skills
- 后续实现应围绕根目录 `claude.md` 继续推进
