# News Digest 设计 Spec

## 产品定义
- **产品名称**：News Digest
- **产品类型**：每日国际要闻中文聚合站
- **核心价值**：一日一期，读完即走
- **技术栈**：Astro 静态站点，零运行时 JS

## 目标受众
- **主要用户**：希望快速了解国际大事的中文读者
- **使用场景**：每天早上花 5-10 分钟浏览国际要闻
- **设备偏好**：移动端优先，桌面端也需要良好体验

## 核心信息与内容板块
1. **顶部导航**：品牌名 + 日期 + 导航链接（首页/归档/关于/RSS）
2. **标题区域**：「每日国际要闻」+ 副标题 + 日期信息
3. **搜索功能**：支持中英文搜索新闻标题、摘要、来源
4. **新闻列表**：20条新闻条目，每条包含：
   - 标题（中文翻译）
   - 摘要（中文翻译）
   - 来源标签（Reuters、BBC等）
   - 发布日期
   - 原文链接
5. **页脚**：项目信息、导航链接、版权声明

## 情感基调与气质关键词
- **可靠**：来自权威媒体的信息来源
- **高效**：快速获取关键信息
- **专注**：无干扰的阅读环境
- **精致**：极简但不简陋

## 输出格式
- **类型**：网页（静态 HTML）
- **尺寸**：响应式，移动端优先
- **断点**：768px（平板）、1024px（桌面）

## 已知约束
- 使用系统字体栈（不引入额外字体文件）
- 零运行时 JS（搜索功能除外）
- 支持深色模式
- Lighthouse 评分 ≥ 95

## 图片需求
- 无（纯文本内容，不需要图片）

## 设计方向

### 方向 A · 经典报纸风格
- **视觉DNA**：衬线字体标题、网格布局、黑白为主、专业感
- **气质**：权威、传统、可信
- **适用场景**：强调新闻的严肃性和专业性

### 方向 B · 现代杂志风格
- **视觉DNA**：大标题、视觉冲击、色块分区、现代排版
- **气质**：时尚、活力、年轻
- **适用场景**：吸引年轻读者，提升视觉吸引力

### 方向 C · 极简升级版
- **视觉DNA**：保持极简核心、精致细节、留白艺术、微妙层次
- **气质**：优雅、克制、精致
- **适用场景**：保持现有风格，提升品质感
